# CreateStructDTO
结构化DTO 是一种`eq`对象关系树的传输对象映射 快捷 易于阅读 的方式来实现快速返回结构化关系数据的一种功能

- 支持按任意属性拉取需要的属性
- 支持任意层级的自定义设置

## 1.创建dto(必须)
手动创建DTO是一件很麻烦的事情,可以在`easy-query:1.10.60^`+`插件0.0.48^`快速生成嵌套结构化对象模型,
在需要创建DTO的包上面进行右键选择`CreateStructDTO`
